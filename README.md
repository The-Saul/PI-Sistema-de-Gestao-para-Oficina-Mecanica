# PI-Sistema-de-Gestao-para-Oficina-Mecanica

Esse será o repositório utilizado para o desenvolvimento do Projeto Integrador: Engenharia de Software, do terceiro semestre dos cursos de Análise e Desenvolvimento de Sistemas e Sistema da Informação.

Este repositório contém o desenvolvimento do Projeto Integrador (PI) da disciplina de Engenharia de Software, do terceiro semestre dos cursos de Análise e Desenvolvimento de Sistemas e Sistemas de Informação.


## Sobre o Projeto

O projeto tem como objetivo desenvolver um sistema web para gerenciamento de oficinas mecânicas, com estrutura semelhante a um ERP (Enterprise Resource Planning), permitindo o controle e organização das principais operações do negócio.
O sistema será utilizado por funcionários da oficina para gerenciar informações de forma centralizada, proporcionando mais eficiência, organização e confiabilidade nos dados.


## Tecnologias Utilizadas

- PHP (Backend)
- PostgreSQL (Banco de Dados)
- HTML, CSS e JavaScript (Frontend)
- XAMPP (Servidor local)
- Git e GitHub (Controle de versão)
- Figma (Prototipação)
- Trello (Gestão do projeto)
